Массивы

Для хранения упорядоченных коллекций существует особая структура данных, которая называется массив, Array.

Объявление

let arr = new Array(); let arr = [];

Элементы массива нумеруются, начиная с нуля. Мы можем заменить элемент: fruits[2] = 'Груша'; .

В массиве могут храниться элементы любого типа. Список элементов массива, как и список свойств объекта, может оканчиваться запятой.

Обращение к последнему элементу массива:

alert(fruits[fruits.length - 1]);
alert(fruits.at(-1)) // добавлено в язык недавно

Методы pop/push, shift/unshift

Очередь:

Стэк:

Массивы в JavaScript могут работать и как очередь, и как стек. Мы можем добавлять/удалять элементы как в начало, так и в конец массива. В компьютерных науках структура данных, делающая это возможным, называется двусторонняя очередь.

fruits.pop() // удаляет последний элемент массива
fruits.push("Груша") // добавляет элемент в конец массива
fruits.shift() // удаляет первый элемент массива
fruits.unshift("Яблоко") // добавляет первый элемент массива

Перебор элементов

  1. for
  2. for .. of
  3. for .. in

Цикл for..of не предоставляет доступа к номеру текущего элемента, только к его значению,

Технически, так как массив является объектом, можно использовать и вариант for..in. Скрытые недостатки этого способа:

  1. Цикл for..in выполняет перебор всех свойств объекта, а не только цифровых.
  2. Цикл for..in оптимизирован под произвольные объекты, не массивы, и поэтому в 10-100 раз медленнее.

Немного о «length»

Свойство length автоматически обновляется при изменении массива. Если быть точными, это не количество элементов массива, а наибольший цифровой индекс плюс один.

Свойстве length – его можно перезаписать. сли мы вручную увеличим его, ничего интересного не произойдёт. Зато, если мы уменьшим его, массив станет короче. Этот процесс необратим. Самый простой способ очистить массив – это arr.length = 0;.

new Array()

Он редко применяется, так как квадратные скобки [] короче. Кроме того, у него есть хитрая особенность. Если new Array вызывается с одним аргументом, который представляет собой число, он создаёт массив без элементов, но с заданной длиной.

Многомерные массивы

Массивы могут содержать элементы, которые тоже являются массивами. Это можно использовать для создания многомерных массивов, например, для хранения матриц

let arr = [1, 2, 3];

alert( arr ); // 1,2,3
alert( String(arr) === '1,2,3' ); // true

alert( [1] + 1 ); // "11"
alert( [1,2] + 1 ); // "1,21"
