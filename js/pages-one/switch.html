Конструкция «switch»

Проверка совпадения выражения a с каждым выражением в case выполняется по правилам алгоритма ===.

switch(x) {
 case 'value1': // if (x === 'value1')
  ...
  [break]

 case 'value2': // if (x === 'value2')
  ...
  [break]

 default:
  ...
 [break]
}

Однако может оказаться, что для ваших целей больше подходит равенство с преобразованием типов, чтобы включить его, необходимо вместо параметра a поставить true:

let a = "42";

switch (true) {
 case a == 10:
  console.log("10 or '10'")
  break

 case a == 42:
  console.log("42 or '42'")
  break

 default:
  // сюда управление никогда не передается
}

Такое решение работает, потому что секция case может содержать любое выражение (а не только простые значения); результат этого выражения будет строго сравниваться с проверяемым условием (true). Так как a == 42 дает true, совпадение будет успешно обнаружено.
