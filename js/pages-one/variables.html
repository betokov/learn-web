Переменные

Переменная

Переменная – это «именованное хранилище» для данных.

Способы объявления переменных:

  • let
  • const
  • var

Различия между var и let:

  • Область видимости переменных var ограничивается либо функцией, либо, если переменная глобальная, то скриптом. Такие переменные доступны за пределами блока. Это происходит потому, что на заре развития JavaScript блоки кода не имели лексического окружения.
  • «var» допускает повторное объявление
  • «var» обрабатываются в начале запуска функции. Благодаря "hoisting" (всплытие, поднятие) все объяления var всплывают в самый верх функции. Объявления переменных «всплывают», но присваивания значений – нет. Это означает, что до присвоения переменной значение будет иметь значение undefined.

Имена переменных

В JavaScript есть два ограничения, касающиеся имён переменных:

  1. Имя переменной должно содержать только буквы, цифры или символы $ и _.
  2. Первый символ не должен быть цифрой.

Тонкости значений в переменных

  • Нельзя отделять имена переменных через "-" (my-name)
  • Переменные с именами apple и APPLE – это две разные переменные.
  • Нелатинские буквы разрешены, но не рекомендуются (我)
  • Нельзя использовать зарезервированные слова

Зарезервированные слова

break, case, class, catch, continue, debugger, default, delete, do, else, export, extends, finally, for, function, if, import, in, instaceof, let, new, return, super, switch, this, throw, try, typeof, var void, while, with, yield

Зарезервированные слова на будущее

enum, await

Зарезервированные слова в non-strict режиме

implements, package, protected, static, interface, private, public

Придумывать правильные имена переменным

Название переменной должно иметь ясный и понятный смысл, говорить о том, какие данные в ней хранятся. Максимально желательная длина названия переменных 3 слова.

Общепринятые подходы:

  • значение — существительное: cat, dog;
  • массив — существительное в множественном числе: cats, dogs;
  • функция — глагол: download, update;
  • класс — существительное с заглавной буквой: Cat, Dog;
  • константа — всё имя пишется заглавными буквами в стиле snake_case: MAXIMUM_FOOD, LIMIT_X;
  • обработчик — имя описывает событие, по которому он вызывается. Как правило, содержит префикс on или постфикс Handler: onButtonClick, formSendHandler.

Стили именования переменных:

  • PascalCase
  • snake_case
  • SCREAMING_SNAKE_CASE
  • kebab-case
