Поиск: getElement*, querySelector*

document.getElementById или просто id

document.getElementById("id").style.color = "red"
id.style.color = "red"

…Но это только если мы не объявили в JavaScript переменную с таким же именем. Но крайне не рекомендуется делать такой поиск.

querySelectorAll

document.querySelectorAll('ul > li:last-child');

Псевдоклассы тоже работают

matches

Метод elem.matches(css) ничего не ищет, а проверяет, удовлетворяет ли elem CSS-селектору, и возвращает true или false.

closest

Метод elem.closest(css) ищет ближайшего предка, который соответствует CSS-селектору. Сам элемент также включается в поиск.

Метод возвращает либо предка, либо null, если такой элемент не найден.

getElementsBy*

  • elem.getElementsByTagName(tag) ищет элементы с данным тегом и возвращает их коллекцию. Передав "*" вместо тега, можно получить всех потомков.
  • elem.getElementsByClassName(className) возвращает элементы, которые имеют данный CSS-класс.
  • document.getElementsByName(name) возвращает элементы с заданным атрибутом name. Очень редко используется.
