Браузерное окружение, спецификации

Что доступно Javascript в браузерном окружении

DOM (Document Object Model

DOM - объектная модель документа, которая представляет все содержимое страницы в виде объектов, которое можно менять.

Объект document - основная «входная точка». С его помощью мы можем что-то создавать или менять на странице.

BOM (Browser Object Model)

Объектная модель браузера - это дополнительные объекты, предоставляемые браузером (окружением), чтобы работать со всем, кроме документа: setTimeout, alert, location, ...

Каждый HTML-тег является объектом. Вложенные теги являются «детьми» родительского элемента. Текст, который находится внутри тега, также являются объектом.

DOM - это представление HTML-документа в виде дерева тегов.

Теги является узлами-элементами. Текст внутри элементов образует текстовые узлы.

Существует всего два исключения из этого правила:

  • По историческим причинам пробелы и перевод строки перед тегом <head> игнорируются
  • Если мы записываем что-либо после закрывающего тега </body>, браузер автоматически перемещает эту запись в конец body, поскольку спецификация HTML требует, чтобы всё содержимое было внутри </body>. Поэтому после закрывающего тега </body> не может быть никаких пробелов.

Есть и некоторые другие типы узлов, кроме элементов и текстовых узлов - узел комментарий
