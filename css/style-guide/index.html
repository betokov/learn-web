Editor Config

Сайт, где прописан полный список параметорв для editor config.

Для того, чтобы VSCODE знал какие нужны отступы и размеры отступов и т.д, для разных файловых расширений, нужно создать в корне проекта файл .editorconfig

root = true // настройка где лежит editor config

[*] // Для всех расширений файлов
end_of_line = lf // конец строки
insert_final_newline = true // вставлять ли последнюю строку
trim_trailing_whitespace = true // удалять ли пробелы в конце строки

[*.{js,py}] // Только для файлов с расширением js, py
charset = utf-8 // кодировка

[*.py]
indent_style = space // тип отступа
indent_size = 4 // размер отступа
tab_width = 4 // ширина таба если используем таб

[Makefile]
indent_style = tab // тип отступа

[lib/**.js]
indent_style = space
indent_size = 2

[{package.json,.travis.yml}]
indent_style = space
indent_size = 2

Если вы работает в команде мало положить его в корень проекта. Нужно убедиться что он есть у всех, кто разрабатывает проект. Можно установить через npm пакет ( editorconfig-checker) , который проверяет соответствуют ли файлы данным правилам которые прописаны в конфиге.

Также можно добавить автотесты при заливке на гитхаб, так называемые «git actions».

Доступное скрытие

Контент скрыт утилитарным классом visually-hidden, чтобы он был доступен для скринридеров и поисковиков.

.visually-hidden {
 position: absolute;
 width: 1px;
 height: 1px;
 margin: -1px;
 border: 0;
 padding: 0;
 white-space: nowrap;
 clip-path: inset(100%);
 clip: rect(0 0 0 0);
 overflow: hidden;
}

<h2 class="visually-hidden">Заголовок</h2>

Порядок слов

Объявления логически связанных свойств сгруппированы в следующем порядке:

  1. Позиционирование
  2. Блочная модель
  3. Типографика
  4. Оформление
  5. Анимация
  6. Разное
.declaration-order {
 /* Позиционирование */
 position: absolute;
 top: 0;
 right: 0;
 bottom: 0;
 left: 0;
 z-index: 100;

 /* Блочная модель */
 display: block;
 float: right;
 width: 100px;
 height: 100px;
 margin: 10px;
 padding: 10px;

 /* Типографика */
 font-family: "Arial", sans-serif;
 font-style: normal;
 font-size: 13px;
 line-height: 20px;
 font-weight: 700;
 text-align: center;
 color: #333333;

 /* Оформление */
 background-color: #f5f5f5;
 border: 1px solid #e5e5e5;
 border-radius: 3px;
 opacity: 1;

 /* Анимация */
 transition: color 1s;

 /* Разное */
 will-change: auto;
}
